# TBC-Logistic-Regression
This project aims to develop a logistic regression model to predict the likelihood of Tuberculosis (TB) occurrence based on clinical, demographic, and environmental risk factors. 
## Objectives 
- Develop a prediction model using **Logistic Regression**.
- Support early TB screening with data-driven insights.
- Identify the most significant risk factors influencing TB detection.

├── src/                 # Source code

└── README.md            # Documentation
